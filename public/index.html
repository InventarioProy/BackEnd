Categoria

Guardar datos POST http://localhost:3000/api/categoria/add 
                Body
                nombre(string-50)
                descripcion(string-255)

                Query
                ?_id=texto
Lista de todos los registros GET http://localhost:3000/api/categoria/list 
                Query

                Body
               _id(string)
                nombre(string-50)
                descripcion(String-255)
Elimina un registro DELETE http://localhost:3000/api/categoria/remove 
                Body
               _id(string)

                Body
               _id(string)
deactivate una Categoria PUT http://localhost:3000/api/categoria/deactivate 
                Body
               _id(string)
